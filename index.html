JqChat

Hey, this is a demo!

This is just a landing page.

JqChat is a jQuery plugin for instant messaging like facebook/gmail chat.
Click on 'Chat (Disconnected)' at the bottom of this page to start using it.
Is made entirely with JavaScript (even the server side).
Built on top off these components -> Jquery UI, Node.js, Socket.IO.
